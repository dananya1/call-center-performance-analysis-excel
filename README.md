Project Overview

This project focuses on analyzing customer call center data using Excel to evaluate customer satisfaction, representative performance, purchase behavior, and call trends. Interactive dashboards were created to generate actionable business insights and improve reporting efficiency.

Tools Used:

MS Excel
Pivot Tables
Power Query
VLOOKUP
Charts & Dashboards

Key Features:

Built interactive dashboards to track customer calls, satisfaction ratings, purchase amounts, and representative performance.
Performed data cleaning and transformation using Power Query and Excel formulas.
Visualized call trends, city-wise performance, and weekday analysis using charts and KPI cards.
Key Insights
Analyzed 1,000+ customer call records across multiple cities and representatives.
Identified Saturday as the peak call day with 161 calls and March as the busiest month with 155 calls.
Discovered Cleveland as the highest-volume city with 389 calls, highlighting performance variation across representatives.
